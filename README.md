# React Native Quiz App

## Overview

The React Native Quiz App is a mobile application developed using React Native, Expo, and Firestore. This app provides an engaging and interactive quiz experience for users. With a user-friendly interface, it allows players to answer a series of quiz questions. The app is built to be easily deployable, thanks to Expo's tools and services.

### Project Motivation

This project was created with the primary goal of helping me learn React Native. The codebase is structured to be beginner-friendly, making it a valuable resource for others interested in learning React Native.
